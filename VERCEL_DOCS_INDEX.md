## ğŸ”‘ Key Concepts

### Vercel Structure
- **Frontend**: React SPA served as static files from `client/build/`
- **Backend**: Express API running on Vercel Functions at `api/index.js`
- **Database**: Firebase Firestore for persistent data
- **Real-time**: Socket.IO with polling fallback

### Environment Variables
```

FIREBASE_PROJECT_ID=...
FIREBASE_PRIVATE_KEY=...
FIREBASE_CLIENT_EMAIL=...
```
